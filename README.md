# Task Manager API

A simple CRUD backend built with Node.js, Express, and MySQL.  
Deployed on Render.

## Features
- Create, read, update, delete tasks
- MySQL database for persistence
- REST API endpoints

## Tech Stack
- Node.js
- Express
- MySQL
- Render (Hosting)

## Setup
```bash
git clone <repo-url>
cd task-manager-api
npm install
